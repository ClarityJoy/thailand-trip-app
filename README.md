# תאילנד 2026 — אפליקציית הטיול שלנו 🇹🇭

אפליקציית מלווה למסע המשפחתי בתאילנד (3–17.7.2026), בהשראת
[Thailand Trip App של רן אביב](https://ranaviv.com/thailand-trip-app).
כל הטיול הפך ל"דאטהבייס" אחד שמזין את כל המסכים: מסלול, מפה, מדריכים,
משחקים לילדים, מזג אוויר, מטבע, כספת מסמכים וחירום.

עברית מלאה (RTL) · עובד גם בלי אינטרנט (PWA) · ניתן להתקנה למסך הבית.

## מה יש באפליקציה

- **בית** — דינמי: ספירה לאחור עד הטיסה, ובמהלך הטיול מציג את היום, העיר, המלון ומזג האוויר.
- **מסלול** — לוח יום-יום של כל הטיסות, המעבורות, המלונות והפעילויות. כל פריט נפתח ב-Google Maps.
- **מפה** — מפה אינטראקטיבית (Leaflet) עם כל המלונות והפעילויות וקו המסלול.
- **מדריכים** — מדריך לכל יעד: היסטוריה, עובדות מפתיעות, טיפים ומה לעשות.
- **טיסות ומלונות** — כל ההזמנות במקום אחד, עם תגיות "מוזמן / לסגור".
- **אטרקציות** — צ'ק-ליסט לילדים לסימון תוך כדי הטיול.
- **תקציב והוצאות** — מעקב הוצאות לפי קטגוריה, יעד תקציב והמרת מטבע.
- **מטבע** — המרת בָּאט לשקלים/דולרים (שערים חיים) + עובדות על הכסף.
- **מזג אוויר** — תחזית 5 ימים לכל יעד (Open-Meteo).
- **כספת מסמכים** — צ'ק-ליסט של דרכונים, ויזה, ביטוח והזמנות — זמין אופליין.
- **חירום** — כל המספרים החשובים בחיוג מהיר.
- **משחקים** — טריוויה, מילון תאי "איך אומרים…?" ומשחק זיכרון, עם בולים ⭐.

## הנתונים שלנו

מסלול: בנגקוק → קו טאו (קורס צלילה Open Water) → קו סמוי → בנגקוק.
כל פרטי הטיסות, המלונות והפעילויות נמצאים בקובץ `app/data/trip.ts` — זה
המקום היחיד לעדכון נתונים, והם מתפשטים אוטומטית לכל המסכים.

## הרצה מקומית

```bash
npm install
npm run dev
```

פתחו [http://localhost:3000](http://localhost:3000).
טיפ: אפשר לדמות תאריך בטיול עם `?date=2026-07-08` כדי לראות את מצב "במהלך הטיול".

## פריסה ל-Vercel

1. דחפו את הריפו ל-GitHub.
2. ייבאו ב-[vercel.com/new](https://vercel.com/new) — Next.js מזוהה אוטומטית.
3. Deploy. אין צורך במשתני סביבה.

## מבנה

- `app/data/trip.ts` — מאגר הנתונים של הטיול (מקור האמת).
- `app/lib/` — תאריכים, נתונים חיים (מזג אוויר/מטבע), שמירה מקומית.
- `app/screens/` — כל המסכים.
- `app/components/` — רכיבי UI ומפה.
- `public/` — manifest, service worker ואייקונים ל-PWA.

## מקורות נתונים חיצוניים (חינמיים, ללא מפתח)

- מפה: OpenStreetMap · מזג אוויר: Open-Meteo · מטבע: open.er-api.com.
  אם אין רשת — הכל ממשיך לעבוד עם נתונים מקורבים/שמורים.
